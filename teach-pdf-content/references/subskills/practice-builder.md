# Practice Builder Subskill

Use this module when creating or updating `02-active-recall.md`, `03-exercises.md`, `04-glossary.md`, `05-review-plan.md`, `06-memory-cards.md`, or tutoring follow-ups.

## Learning Objectives

Write each objective as observable performance:

`LOx: The learner can <action> <content> under <condition>; mastery level: <recognize/explain/distinguish/sequence/apply/memorize/diagnose>; mastery means <criterion>; assessed by <question/exercise/card id>; source: <page/section>.`

Avoid vague goals like "understand" unless followed by concrete criteria.

## Recall And Exercise Rules

- Keep question sections answer-free except for short hints.
- Put complete answers in final answer-key sections.
- Map every recall prompt and exercise to one or more objectives.
- Include source basis or mark `teacher-designed prompt, not a source example`.
- Include common wrong answers and scoring/self-check points.

Exercise fields:

- Objective
- Type: recall | discrimination | procedure | error-correction | transfer
- Difficulty: Basic | Core | Transfer
- Prompt
- Hint
- Expected answer
- Scoring points
- Common wrong answer
- Source basis

## Balanced Practice Set

For a normal chapter, include:

- definition recall
- concept discrimination
- procedure or calculation
- error correction
- transfer to a new case
- code or implementation task when relevant

Transfer exercises must use a scenario beyond the source example.

## Glossary

Each glossary term should include:

- definition
- intuition
- source
- example
- non-example or confusion
- symbols/formulas when relevant

## Memory Cards

Create `06-memory-cards.md` only after objectives and key concepts are clear. Card only high-retrieval items:

- exact definitions expected closed-book
- stable formulas or conditions
- process steps
- canonical contrasts
- repeated traps
- short "why this design exists" answers

Do not card long prose, broad motivation, or full examples.

## Review Plan

Tie review tasks to objectives, weak points, and wrong-answer recovery:

- today: first closed-book check and immediate corrections
- tomorrow: recall without notes
- three days later: mixed exercises and mistakes
- one week later: transfer and oral review

If a learner misses the same idea twice, change representation: analogy, diagram, formula walk-through, boundary case, or error correction.
